# Resource Allocation System

## Overview

The Resource Allocation System is a full-stack web application developed to efficiently manage employees, projects, and resource allocation. It uses AI-powered recommendations to assign employees to projects based on their skills and workload.

---

## Features

- Employee Management
- Project Management
- AI-based Resource Recommendation
- Skill Gap Analysis
- Workload Analysis
- RESTful APIs with Spring Boot
- Responsive React Frontend
- MySQL Database Integration

## Installation

### Clone the repository

```bash
git clone https://github.com/Samruddhi0210/resource-allocation-system.git
```

### Start the Frontend

```bash
cd frontend
npm install
npm start
```

### Start the Backend

```bash
cd resource-allocation-system
mvn spring-boot:run
```

### Start the AI Service

```bash
cd python-ai
pip install -r requirements.txt
python app.py
```
